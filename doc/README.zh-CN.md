# SUMA Cloud-HFS

### Cloud HTTP File Sharing / 云端 HTTP 文件分享工具

SUMA Cloud-HFS（Cloud HTTP File Sharing 的缩写）是免费、无需帐号即可使用的 Windows 桌面文件分享服务，可在局域网中分享文件，或透过 Cloudflare Tunnel 建立临时的远程下载网址。由 [域创数位工作室 (LOCALSOFT Digital Studio)](https://suma.tw) 开发。

透过 Cloudflare 远程安全通道（内网穿透），**不需要注册帐号，也不需要 Cloudflare 帐号**即可建立一个临时的公开下载链接，让身处不同网络环境（非本机局域网）的远程用户直接下载您所分享的本机文件。此技术可完美穿透防火墙与宽带内部网络（NAT/CGNAT），无需固定公网 IP 亦不需进行 any 连接端口对应与转发设定。

---

## 快速开始

1. 开启 Cloud-HFS。
2. 在「分享管理」加入文件夹或文件。
3. 回到仪表板并启动分享服务。
4. 将显示的网址或 QR Code 提供给下载者。

*如需跨网络分享，请开启「Cloudflare 分享」并启动远程分享。*

---

## 简介

Cloud-HFS 提供图形化界面管理 HTTP 文件分享服务。你可以设定站台名称、连接端口、HTTP 分享服务 Logo、下载限速与断点续传，并汇入或汇出设定及分享项目。侧边栏即时显示目前连接数与下载流量，让分享状态一目了然。

**Web 优先跨平台架构重构：**
本次重构以 Web 界面为核心，将桌面壳层聚焦于系统整合与原生能力，并纳入 Windows、Linux 与 macOS 的平台差异考量，为后续跨平台软件开发与扩充建立更稳固的基础。

## 功能特色

### 📂 分享管理
- Windows 桌面图形化界面。
- 分享整个文件夹、指定文件、连结网址，并支持多选批次新增。
- 多层级的虚拟目录管理，弹性组织分享结构。
- 支持汇入与汇出设定及分享项目。

### 🌐 远程分享
- 显示区域网络存取网址与 QR Code。
- 服务预设绑定区网 IP，并支持 Windows 离线环境 fallback 侦测。
- 透过 Cloudflare Tunnel 建立临时远程分享网址（**免费、免注册、免 Cloudflare 帐号**）。
- 按需取得缩短网址（`suma.tw`）。

### 📥 下载体验
- 支持下载限速（总限速 / 个人限速动态分配）与 HTTP Range 断点续传。
- 可设定下载密码锁，保护分享内容。
- 可自订下载页面的 HTTP 分享服务 Logo。
- 下载页面支持桌面、平板与手机版面。
- 下载页面「回到顶端」浮动按钮，改善长列表浏览体验。

### ⚙️ 系统与设定
- 侧边栏即时显示目前连接数与下载流量。
- 记录下载次数、流量与下载记录。
- 支持繁体中文、简体中文与英文界面，首次启动自动侦测系统语系。
- 启动时可依每天、每周或每月频率检查新版本。

### 💻 Web 优先跨平台架构重构
- 新增分享项目时会依 Windows、Linux 与 macOS 的系统及敏感路径规则进行保护，降低误分享风险。
- 完善 Tauri Mock 测试与 API 行为处理，提升跨平台功能验证的可靠性。
- 调整桌面 App 的窗口初始化与原生错误讯息处理，让各平台的应用程序整合更一致。

## Beta 测试版本

Beta 测试版本可能设有使用期限，目的不是作为付费门槛，而是让使用者在测试期间持续取得较新的版本，协助回报问题并改善稳定性、安全性与整体使用体验。期限到期时，程序会保留汇出设定功能，方便使用者保存既有设定后更新至新版本。

---

## 版本检查

前往「基本设定」的「检查更新」，选择每天、每周或每月。预设为每周，程序会在启动时依设定频率读取 SUMA 官方版本列表；发现新版本时会显示更新内容与下载按钮。

变更检查频率并储存设定后，Cloud-HFS 会立即执行一次版本检查。检查失败不会影响文件分享服务。
