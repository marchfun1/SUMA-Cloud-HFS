# SUMA Cloud-HFS

### Cloud HTTP File Sharing / 雲端 HTTP 檔案分享工具

SUMA Cloud-HFS（Cloud HTTP File Sharing 的縮寫）是免費、免帳號即可使用的 Windows 桌面檔案分享服務，可在區域網路中分享檔案，或透過 Cloudflare Tunnel 建立臨時的遠端下載網址。由 [域創數位工作室 (LOCALSOFT Digital Studio)](https://suma.tw) 開發。

透過 Cloudflare 遠端安全通道（內網穿透），**不需要註冊帳號，也不需要 Cloudflare 帳號**即可建立一個臨時的公開下載連結，讓身處不同網路環境（非本機區域網路）的遠端用戶直接下載您所分享的本機檔案。此技術可完美穿透防火牆與寬頻內部網路（NAT/CGNAT），無需固定公網 IP 亦不需進行 any 連接埠對應與轉送設定。

---

## 快速開始

1. 開啟 Cloud-HFS。
2. 在「分享管理」加入資料夾或檔案。
3. 回到儀表板並啟動分享服務。
4. 將顯示的網址或 QR Code 提供給下載者。

*如需跨網路分享，請開啟「Cloudflare 分享」並啟動遠端分享。*

> 💡 **Linux CLI 伺服器版本**：如果您是在 Linux 伺服器、NAS 或無頭環境下運行，請參閱 [Linux CLI 版本安裝與使用說明](INSTALL_Cloud-HFS_LINUX_CLI.zh-TW.md)。

---

## 簡介

Cloud-HFS 提供圖形化介面管理 HTTP 檔案分享服務。你可以設定站台名稱、連接埠、HTTP 分享服務 Logo、下載限速與斷點續傳，並匯入或匯出設定及分享項目。側邊欄即時顯示目前連線數與下載流量，讓分享狀態一目瞭然。

**Web 優先跨平台架構重構：**
本次重構以 Web 介面為核心，將桌面殼層聚焦於系統整合與原生能力，並納入 Windows、Linux 與 macOS 的平台差異考量，為後續跨平台軟體開發與擴充建立更穩固的基礎。

## 功能特色

### 📂 分享管理
- Windows 桌面圖形化介面。
- 分享整個資料夾、指定檔案、連結網址，並支援多選批次新增。
- 多層級虛擬目錄管理，彈性組織分享結構。
- 支援匯入與匯出設定及分享項目。

### 🌐 遠端分享
- 顯示區域網路存取網址與 QR Code。
- 服務預設繫結區網 IP，並支援 Windows 離線環境 fallback 偵測。
- 透過 Cloudflare Tunnel 建立臨時遠端分享網址（**免費、免註冊、免 Cloudflare 帳號**）。
- 按需取得短網址（`suma.tw`）。

### 📥 下載體驗
- 支援下載限速（總限速 / 個人限速動態分配）與 HTTP Range 斷點續傳。
- 可設定下載密碼鎖，保護分享內容。
- 可自訂下載頁面的 HTTP 分享服務 Logo。
- 下載頁面支援桌面、平板與手機版面。
- 下載頁面「回到頂端」浮動按鈕，改善長列表瀏覽體驗。

### ⚙️ 系統與設定
- 側邊欄即時顯示目前連線數與下載流量。
- 記錄下載次數、流量與下載紀錄。
- 支援繁體中文、簡體中文與英文介面，首次啟動自動偵測系統語系。
- 啟動時可依每天、每週或每月頻率檢查新版本。

### 💻 Web 優先跨平台架構重構
- 新增分享項目時會依 Windows、Linux 與 macOS 的系統及敏感路徑規則進行保護，降低誤分享風險。
- 完善 Tauri Mock 測試與 API 行為處理，提升跨平台功能驗證的可靠性。
- 調整桌面 App 的視窗初始化與原生錯誤訊息處理，讓各平台的應用程式整合更一致。

## Beta 測試版本

Beta 測試版本可能設有使用期限，目的不是作為付費門檻，而是讓使用者在測試期間持續取得較新的版本，協助回報問題並改善穩定性、安全性與整體使用體驗。期限到期時，程式會保留匯出設定功能，方便使用者保存既有設定後更新至新版本。

---

## 版本檢查

前往「基本設定」的「檢查更新」，選擇每天、每週或每月。預設為每週，程式會在啟動時依設定頻率讀取 SUMA 官方版本清單；發現新版本時會顯示更新內容與下載按鈕。

變更檢查頻率並儲存設定後，Cloud-HFS 會立即執行一次版本檢查。檢查失敗不會影響檔案分享服務。
